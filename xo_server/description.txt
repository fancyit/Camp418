Сущности:
  1. Игрок
  2. Игра  
Описание сущностей:
  Игрок
    Поля:
      ИД - guid
      Логин - строка
      Пароль - строка
      Почта - строка
      Текущая Игра - guid
      История Игр - массив из guid- ов прошедших игр
      Друзья - массив из guid ов других пользователей
  Игра
    Поля:
      ИД -guid
      Игровое поле - массив int(3,3) или одномерный(задача на пересмотр определения состояния игрового поля)
      Владелец - guid(игрока)
      Соперник - guid(игрока)
      Зрители - массив guid -ов
      Победитель - guid(игрока)
      Игра закончена - булево
      Игра на паузе - булево
